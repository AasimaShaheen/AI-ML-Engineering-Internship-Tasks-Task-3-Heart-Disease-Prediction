# AI-ML-Engineering-Internship-Tasks-Task-3-Heart-Disease-Prediction
Task Objective: Build a model to predict whether a person is at risk of heart disease based on their health data.
Dataset Used : Heart Diseases  
Model Applied : Logistic Regression &amp; Decision Tree 
Results : The dataset was cleaned and EDA identified key indicators of heart disease. 
Logistic Regression showed strong performance with high accuracy and AUC.
Chest pain type, heart rate, and ST depression were the most important features.
